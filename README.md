# VibeRender — пересобранная версия

Рабочая кодовая база VibeRender, собранная по итогам аудита прототипа
(preview--vibe-render-net.lovable.app). Архитектурные проблемы исходного MVP
решены на уровне фундамента: мультиязычность (EN/RU/AR), RTL для арабского,
консистентный glassmorphism, готовность к Web3.

**Статус: раздел P0 из `BUILD_BRIEF.md` выполнен** — все 4 страницы собраны,
проект запускается.

---

## Как запустить на своём компьютере

Нужен установленный Node.js 18+ (nodejs.org). Затем в папке проекта:

```bash
npm install     # установить зависимости (один раз)
npm run dev      # запустить — откроется http://localhost:5173
```

`npm run build` — собрать продакшн-версию в папку `dist`.

> Зависимости не установлены заранее: в среде, где собирался проект, не было
> доступа к npm-реестру. `npm install` нужно выполнить у себя — это штатный шаг.
> Проще всего: открыть папку в Claude Code и попросить «выполни npm install и
> запусти npm run dev».

Перед первым запуском скопируйте `.env.example` в `.env.local` и впишите
бесплатный `VITE_WC_PROJECT_ID` с https://cloud.reown.com (нужен для окна
подключения кошелька; без него запустится, но кнопка кошелька будет неполной).

## Что чинит эта версия (из аудита)

- **Языки и RTL** — EN/RU/AR в фундаменте; layout не ломается при удлинении
  текста, зеркалится для арабского.
- **Сети** — раньше сайт называл то «Polygon · zkSync · Base», то «Base ·
  Arbitrum»; теперь единый источник `chains.ts`.
- **Доска задач операторов** — 6-колоночная таблица ломалась на телефоне;
  теперь адаптивная сетка без горизонтального скролла.
- **Glassmorphism** — единый стеклянный стиль для всех карточек.
- **Web3** — кнопки Connect Wallet / Lock Escrow / Withdraw имеют состояния и
  готовы к интеграции контрактов.
